## ğŸ¨ Customization

### Colors
The color scheme can be easily customized by modifying the CSS variables in `css/style.css`:

```css
:root {
    --primary-color: #2563eb;
    --secondary-color: #1e40af;
    --accent-color: #f59e0b;
    --text-dark: #1f2937;
    --text-light: #6b7280;
}
```

### Content
- Update text content directly in the HTML files
- Modify programs, team members, and impact statistics as needed
- Replace placeholder contact information with actual foundation details
- Update social media links in the footer

### Images
- Team member images use SVG placeholders with initials
- Replace with actual photos by updating the `src` attributes in `who-we-are.html`
- Add program photos and success stories to enhance engagement
